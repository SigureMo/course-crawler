# FAQ

Q1:

我的登录信息输错了（失效了），怎么重新填写？

A1:

重新启动程序，启动时添加参数 -c 并输入新的 Cookies

Q2:

我遇到了一个课程无法成功获取，最快捷的反馈方式是？

A2:

依次进行如下检查：

  • Cookie 是否失效，如果失效请使用参数 -c 并重新输入
  • 当前账号是否加入了该课程，并对该课程有访问权限（比如该学期是否是开启状态，课程是否是付费才能观看）
  • Github issues 中是否有相似问题与解决方案

如果仍然无法解决，请在 Github 提出 issue ，或者邮件联系我，我会尽快处理

Q3:

我想看原版文档

A3:

请前往 Foair 的文档 查看

Q4:

如何参与文档的修改？

A4:

点击文档左下角的“在 GitHub 上编辑此页” 即可~
